Description

The repository contains the code for the live demo/workshop held on May 6th 2016, at ng-conf 2016 extended in Pakistan organized by Attribe Solutions The demo contains a simple application based on Angular2 which is actually a puzzle containing 4 inputs. Each input can have either a value of 0 or 1. The user has to guess the value for each input and solve this puzzle. Once it is solved, the user sees a message of congratulations.
